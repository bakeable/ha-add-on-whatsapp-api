# Icon and Logo Requirements

Create these two PNG images for the add-on:

## icon.png
- **Size**: 128x128px (square)
- **Format**: PNG
- **Used in**: Add-on store, sidebar

## logo.png
- **Size**: 250x100px (recommended)
- **Format**: PNG  
- **Used in**: Add-on details page header

## Design Tips
- Use WhatsApp green (#25D366)
- Use Home Assistant blue (#03A9F4)
- Keep it simple and clear
- Test readability at small sizes

## Quick Creation Tools
- [Canva](https://www.canva.com) - Easy online editor
- [GIMP](https://www.gimp.org) - Free desktop software
- [Figma](https://www.figma.com) - Professional design tool

Once created, place both files in the `evolution_api/` directory.
